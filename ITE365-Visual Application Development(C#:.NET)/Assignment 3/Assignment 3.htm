1 Overview

This assignment provides you, Oh Most Noble Student, the opportunity to use your C# programming skills to use a bit of data to add to the conversation about the "Most Important Topic: Alabama or Auburn Football".

2 Background

The attached data file is an Excel CSV file that contains data about the won-loss records of the football teams from the University of Alabama and Auburn University since 1958 (the year that Bear Bryant was hired as coach at the University of Alabama). The data is organized by row with the name of the team, year, wins, losses, and ties.

3 Problem 1: The Baseball Players Browser

Use WPF to write an application that provides a scrollable list of the team's record for each team in a year. Your UI should look something like this:

Year Alabama Auburn
1958 w-l-t w-l-t

Add two buttons to the end of the list: (1) a button that when clicked will compare the records for the currently selected year and display a message saying whether UA or AU was better in that year, and (2) a button that exits the application when pressed.

4 Submitting your program

Please combine the source code and a screen shot of the output from each solution to these problems into a single PDF document. Your submission document must be in PDF format; submission of documents in any other format will result in deduction of points from your grade. Attach your submission to the assignment entry in Blackboard.
